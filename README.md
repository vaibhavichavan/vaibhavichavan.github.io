## 🚀 Deploy to GitHub Pages

1. Create a new GitHub repository (e.g. `vaibhavi-portfolio`)
2. Upload `index.html` and `README.md`
3. Go to **Settings → Pages**
4. Under **Source**, select `main` branch and `/ (root)` folder
5. Click **Save**

Your site will be live at:
```
https://<your-github-username>.github.io/vaibhavi-portfolio/
```

---

## 🛠 Run Locally

Just open `index.html` in any browser — no server needed.

```bash
open index.html    # macOS
start index.html   # Windows
```

## 🎨 Customisation

| What to change | Where |
|---|---|
| GitHub/LinkedIn links | Search for `github.com/vaibhavichavan2029` and `linkedin.com/in/vaibhavi-chavan` |
| Phone number | Hero section contact area |
| Project GitHub links | Each `<a href="...">` in the Projects section |
| Skill percentages | `data-width` attributes on `.skill-fill` divs |
| Colors | CSS variables at the top of `<style>` block |
| Profile photo | Replace the `.hero-initials` div with an `<img>` tag |
